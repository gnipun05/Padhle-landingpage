FEATURES

  • Tasks
  • Group Session

Goal Setting

Like any self improving process, everything starts with setting your goals and committing to them

Swipes

You can swipe right to mark the task as completed and left to delete it

Predicted Time

Utilizing supervised learning, our app predicts time for task completion based on past data

Add Task

You can add new task by giving the task a difficulty tag which updates the predicted time

Create Session

You can create a session and share the session ID among your friends to see each other's progress

Join Session

You can join active sessions by typing the session ID

Tasks

You can see both active and completed tasks of each user who has joined the same session as you

Bar Graph

Bar graph shows number of Tasks completed for each difficulty tag

Line Chart

Line chart represents data of number of tasks completed daily for last 7 days.

Pie Chart

Pie chart shows the time spend of each completed and active task for current day
