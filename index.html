FEATURES

  • Tasks
  • Group Session
  • Statistics

Goal Setting

Like any self-improving process, everything starts with setting and committing to one's goals.

Swipes

Swipe right to mark the tasks as completed and left to delete them.

Predicted Time

The App uses supervised learning to estimate the time for Task Completion.

Add Task

With the new Tasks being added to the list, the Expected time changes depending on their difficulty.

Create Session

You can create a session and share the session ID among your friends to see each other's progress.

Join Session

One can join the active sessions by entering the respective Session IDs.

Tasks

One can see both active and completed tasks of each user present in the same session.

Bar Graph

The Bar Graph represents the number of Tasks completed daily, corresponding to each difficulty tag.

Line Chart

The Line Chart displays a 7-day history of the number of Tasks that were completed daily.

Pie Chart

The Pie Chart shows the percentage of the current day's time spent on each completed and active task.
